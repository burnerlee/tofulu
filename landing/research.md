# Competitor Research - TOEFL Preparation Platforms

## StudyGround.ai
**Key Pitch Patterns:**
- Celebrates user milestones ("Thank you to our 2,000 users")
- Emphasizes "2026 New TOEFL Mock Test Released!" - highlighting latest format updates
- Offers free mock tests + AI scoring as primary value proposition
- Multiple incentive cards: Free Access, 1:1 Sessions, Discounts
- Clear CTA: "Sign up and get 2 Free Mock Tests + AI Scoring"
- Features highlighted: Unlimited AI Scoring, Super Sets, Authentic Test Experience

**Design Approach:**
- Clean, minimal design with light backgrounds
- Vibrant accent colors (orange, lime green) for CTAs
- Card-based layout for offers
- Centered hero section

## BestMyTest.com
**Key Pitch Patterns:**
- Direct headline: "Taking an Online TOEFL Practice Test"
- Simple value proposition: Free sample test
- Clear 3-step guide mentioned
- Immediate CTA: "START TOEFL SAMPLE TEST" with play icon
- Focus on practice tests and self-study

**Design Approach:**
- Blue hero section with illustration
- Simple navigation
- Direct, action-oriented messaging

## TestGlider.com
**Key Pitch Patterns:**
- Focus on "2026" version updates
- Mock tests as primary offering
- "Start for Free" CTA
- Emphasizes real-time test experience
- Study tips and guidance

**Design Approach:**
- Clean interface
- Tab-based navigation (2025/2026 versions)
- Minimal, focused design

## Common Patterns Across All Competitors:
1. **Free trial/mock test** as primary hook
2. **2026 TOEFL format** updates emphasized
3. **AI scoring** and feedback highlighted
4. **Clear, action-oriented CTAs** (Start Free, Sign Up, Try Now)
5. **Minimal, clean designs** with ample white space
6. **Feature lists** showing value proposition
7. **Social proof** (user counts, testimonials, guarantees)

## Recommended Pitch for Testino:
- Emphasize free access to mock tests
- Highlight 2026 TOEFL format preparation
- Simple, direct messaging about what Testino offers
- Clear, centered CTA button
- Minimal design focusing on core value proposition
